Info

The site dock allows you to install and view static sites /web-apps.

Install from local archive

In case you have stored app-archives (zip-files) on your local disk, you can use this function to install them. Select the archive and select "Open". This will install the app, and it will appear in the app-list.

Running apps

To run an from the list, click on it. It will start in a separate window. To exit fullscreen or close the app, you can use the app-window close button or press "Escape" on the keyboard.

Remove a web-app from the list

To remove apps from the list, select "delete" on the app to be removed. A confirmation dialog will appear, and once confirmed, the app will be be removed. It can be re-installed later.

App-information

If the web-app has an information page included, the info button will show this.

Settings

On the settings page, you can configure:

  • Start apps in fullscreen: when checked, the apps will be launched in full-screen mode
  • Auto-start app: will run automatically when the site dock is started

Notes

When re-installing an app with the same name as an app that was already installed, the previous version will be overwritten.
